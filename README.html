Редактирование шаблона

При работе с шаблоном необходимо редактировать файлы .html, .css, .js

Редактирование файлов HTML и CSS

Редактируемый файл HTML находится в папке "build" шаблона (index.html).

Чтобы открыть файл .html при помощи программы Sublime Text 3, щелкните правой кнопкой мыши по файлу и в контекстном меню выберите Открыть с помощью > Sublime Text 3. Когда файл открыт, его можно редактировать.


Файлы CSS используются для изменения внешнего вида шаблона, например шрифтов (семья шрифтов, размер, цвет и т.п.), фон, значение ширины колонки и т.д. Файлы css можно редактировать при помощи программы Sublime Text 3.


Туториалы по редактированию

Как редактиовать текст

Шаблоны основаны на HTML, поэтому весь контент находится в файлах .html.

  1. Перейдите в папку build шаблона.
  2. Откройте файл index.html в редакторе HTML.
  3. Можно использовать функцию поиска CTRL+F , чтобы найти нужный текст.
  4. Измените текст в редакторе HTML. Сохраните файл и откройте в браузере, чтобы увидеть изменения.

Также, для поиска нужных блоков, можно использовать браузерные инструменты разработчика.


Как редактировать изображения

Перейдите в папку 'build/images' шаблона. Вы можете загружать изображения с тем же названием и расширением, что и исходные, таким образом заменяя их.

Также можно загружать пользовательские картинки со своим названием и расширением. Затем необходимо изменить название файла изображения в html.

  1. Откройте файл .html в папке “site” редактором html.
  2. При помощи функции поиска CTRL+F найдите изображение в файле html.
  3. Отредактируйте название и расширение файла-картинки. Сохраните файл и откройте в браузере, чтобы увидеть изменения.

Если решили пользоваться папкой sources, то папка build вам не нужна (она будет перезаписываться, каждый раз после внесения изменений в файлы и запуска gulp) перед стартом проекта необходимо:

  1. Распаковать архив
  2. Все папки и файлы перенести в директорию проекта
  3. Убедитесь, что установлены node.js и npm. Для этого достаточно написать в терминале:
    • node -v
      Если Вы видите версию, например, v8.11.1, тогда все нормально. Если отсутствует, тогда скачайте последнюю версию
    • npm -v
      Если Вы видите версию, например, 5.6.0, тогда все нормально Если отсутствует, тогда введите в консоле git bash или PhpStorm команду - npm init
  4. Убедиться, что установлен gulp последней версии. Если он не установлен, тогда устанавливаем:
    • npm install gulpjs/gulp#4.0 -g
  5. Инструкция для старта проекта
    • Запустить команду npm install в терминале. Данная команда установит все пакеты, которые указаны в файле package.json, а также все их зависимости

После проделанных выше шагов проект готов к использованию

Можете вносить свои изменения в файлы и после этого запустите gulp, чтобы увидеть изменения используя одну из команд:
  • gulp
  • npm run gulp
  • npm start

    • (После этого создаться папка build)

или запустить gulp сначала и после внесения изменений в файлы, модуль browser-sync автоматически обновит страницу

Также вы можете добавлять свои модули нужные вам для проекта используя команду

  • npm install {name module} --save-dev (так он установит модуль и запишет его в файл package.json)

Туториалы по редактированию

  • Добавление новых блоков
    1. Создать файл в папке templates/sections - name.pug
    2. Добавить его в файл index.pug в тег body при помощи команды include/sections/name_file
  • Добавление новых стилей блоков
    1. Создать файл в папке styles/sections - name.scss
    2. Добавить его в файл main.scss при помощи команды @import sections/name_file
  • Добавление новых js
    1. Создать файл в папке js - name.js
    2. Добавить его в файл gulpfile.js в таск "JS" в массив return gulp.src[...]
  • Добавление новых картинок описано в "Как редактировать изображения"
  • Добавление новых иконок
    1. Добавить иконку в папку icons
    2. Прописать class иконки в файле.pug (например, .icon.icon-style
    3. Открыть файл icons.scss (styles/global/icons.scss)
    4. Добавить стили к класу. Например,
      .icon-style {
      @include sprite($style);
      свои стили;
      }
