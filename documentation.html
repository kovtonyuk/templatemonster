Верстка лендинга

В архиве содержится 2 папки: "screenshots" и "build", и 2 файла: documentation.html и README.html. Также вы увидите архив "sources.zip" который нужно распаковать.

    Папка sources содержит исходные файлы темы:

  • templates
  • style
  • js
  • images
  • fonts
  • gulpfile.js
  • package.json
  • package-lock.json

    • Она защищена паролем 11223344g

    После распаковывания шаблон будет состоять из 3 папок: "screenshots", "build" и "sources" и 2 файла: documentation.html и README.html.

Программные требования:

Sublime Text 3 (скачать бесплатно)или PhpStorm (платный) или любой другой редактор кода. Используется для редактирования файлов:

  • HTML
  • CSS
  • PHP
  • JS

Структура папок и файлов шаблона выглядит следующим образом:

  • documentation.html - Общие сведения
  • README.md - содержит документацию по редактированию и установке шаблона
  • screenshots - содержит скриншоты шаблона
  • build - содержит файлы готового шаблона
    • index.html - содержит разметку сайта
    • send.php - файл обработчик формы отправки вопросов
    • CSS - содержит минимизированный .css, используемый в дизайне
    • JS - содержит минимизированный .js, используемый в дизайне
    • images - содержит все изображения, используемые в файле .html
      • icons - содержит все иконки, используемые в файле sprite.png
    • fonts - содержит все шрифты, используемые в файле .html
  • source - содержит исходные файлы темы
    • fonts - содержит все шрифты
    • images - содержит все изображения
      • icons - содержит все иконки, используемые в файле sprite.png
    • js - содержит все .js, используемые в дизайне
    • style
      • global - содержит базовые .scss, используемые в дизайне
      • sections - содержит все .scss, используемые в дизайне
      • main.scss - содержит подключение всех .scss файлов
    • templates
      • sections - содержит все .pug, используемые в дизайне
      • index.pug - содержит подключение всех .pug файлов
    • send.php - файл обработчик формы отправки вопросов
  • gulpfile.js - содержит подключение gulp модулей и их задачи для обработки файлов
  • package.json - содержит информацию об установленных модулях и информацию о файле
  • package-lock.json - содержит информацию об версиях установленных модулях
  • password - содержит информацию о пароле к архиву source.zip
